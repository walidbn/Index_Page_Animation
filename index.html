Créez votre propre site web

Le parcours web, c'est quoi?
le parcours web permet à l'étudiant d'apprendre à déveloper son propre site web et de le mettre en ligne.
Le programme trés bien à ceux qui ont des idées de sites web et qui veulent passer à l'action et caractériser leurs iddées.
Les technologies utiliées dans ce parcours sont les technologies javascript (React,Node,..).
Ces technologies sont de plus en plus utilisées dans le web et permettent de developper des sites web trés intuitifs et user-freindly.
Comment suivre ce parcours
Ce parcours peut etre suivi dans le cadre du programme PART TIME CODIN G ou SUMMER ACADEMY Selon la disponibilité de l'étudiant.
Les niveauxdu parcours Web
1
Vous allez apprendre les fondamentaux du HTML5 CSS3 et JavaScript, ainsi que des projets a livrer dans des deadlines pour simuler la vie proffisiennel du developpeur.
2
Le niveaux 2 contient des notions plus avvancées comme le framework front-end React Js et de Back-end Nodes.Js et permet a l'étudiant de crés un site web dynamique et intuitif.
3
Le niveaux 3 est le niveaux ou on peut expoloiter les conaissances aquise des les 2 niveaux precedents a travers un projet entier plus complexe et scalable.
